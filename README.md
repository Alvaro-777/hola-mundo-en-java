# hola-mundo-en-java

El prograna “Hola mundo" simplenente muestra "Hola mundo” por pantalla. Es muy sencillo y suele
ser el programa que realiza alguien que quiere aprender un nuevo lenguaje de progranación.

## Compilación y ejecución del programa
para **compilar** el programa teclea lo siguiente (es necesario tener el *J0K*):

console,
javac HolaMundo. java

Y para **ejecutarto**

«console
java HolaMundo
